Development of Design Tools and Application of IoT for Condition Monitoring of PV Systems

In this project, end-to-end solutions were proposed for consumers and investors to encourage PV installations. A User interactive interface was developed for optimaland integrated economical and technical design of various PV system models, that aids a consumer interested to invest in a PV system. Further, A novel test benchwas developed and a Failure Modes & Effect Analysis (FMEA) was performed onStandalone PV systems. From this analysis, an expert system was developed for Fault Diagnosis of Standalone PV systems. Finally, an IoT Based Cost Effective Sensor Network was implemented on small-scale PV rooftop, that ensures reliable operation of the installed panels. The user interactive solutions developed and the simplicity of the IoT network proposed enhance the viability of scaling them up to products, that can serve as a value added service to the consumers.
